# HourMind - 个人待办事项 产品设计 PRD

## 1. 功能概述

**功能名称**：个人待办事项（Personal Tasks / Todo）  
**模块定位**：HourMind 的智能任务管理中心  
**核心目标**：将 AI 能力深度融入个人任务管理，让待办事项不仅仅是列表，而是**智能、可执行、会思考**的任务助手。

**Quantum Glass 风格适配**：使用玻璃拟态卡片、量子青 `#00E5D8` 强调色，保持与聊天界面一致的高端科技感。

---

## 2. 用户价值

- 快速记录任务（自然语言输入）
- AI 智能拆解、建议、总结
- 任务与知识库、对话记录深度关联
- 减少手动管理负担，成为真正的“智能助理”

---

## 3. 核心功能需求

### 3.1 任务基础管理
- 创建任务（支持自然语言，如“下周三前完成产品PRD”）
- 编辑、删除、完成、归档
- 字段包括：标题、描述、优先级（高/中/低）、截止日期、标签、子任务
- 重复任务支持（每日、每周、自定义）

### 3.2 视图模式
- **列表视图**（默认）
- **看板视图**（Kanban - 待办、进行中、已完成）
- **日历视图**（按日期展示）

### 3.3 AI 智能能力（核心差异化）
- **智能创建**：对话中可直接说“把这件事加入待办”
- **任务拆解**：选中大任务 → AI 自动生成子任务
- **每日/每周总结**：AI 自动生成进度报告和建议
- **智能提醒**：结合时间和优先级，在聊天界面主动提醒
- **进度预测**：基于历史习惯预测完成概率
- **关联建议**：自动推荐关联知识库内容或历史对话

### 3.4 筛选与搜索
- 快速筛选：今日、明日、本周、已逾期、高优先级
- 全局搜索 + 标签过滤
- 排序方式：截止日期、优先级、创建时间

---

## 4. 页面布局设计（Quantum Glass）

**整体布局**：
- **左侧导航**：与主系统一致（可悬停呼出）
- **顶部栏**：标题“个人待办事项” + “+ 新建任务”量子青发光按钮 + 搜索框 + 视图切换（List / Kanban / Calendar）
- **中间主区域**：任务列表（玻璃卡片形式）
- **右侧面板**（可折叠）：选中任务的详细信息

**任务卡片设计**：
- 左侧：完成 Checkbox（量子青）
- 中间：标题 + 描述预览
- 右侧：优先级标签（颜色区分）、截止日期、标签
- Hover 效果：卡片上浮 + 辉光增强

**右侧详情面板**：
- 任务标题
- 详细描述（支持 Markdown）
- 子任务清单（可勾选）
- 关联知识库 / 对话记录
- AI 建议区（“AI 推荐”玻璃卡片）
- 操作按钮：编辑、删除、移至知识库

---

## 5. 交互流程

**主要流程**：
1. 点击左侧导航 → 进入个人待办事项
2. 点击“+ 新建任务” → 弹出玻璃输入框（支持语音输入）
3. 选中任务 → 右侧自动展开详情 + AI 建议
4. 在聊天界面中说“把刚才讨论的需求转为任务” → 自动创建并关联

**快捷操作**：
- Ctrl + T：快速新建任务
- 在任意页面输入“待办：xxx” → 快速添加

---

## 6. 与其他模块的关联

- **聊天模块**：支持直接从对话创建任务、关联任务
- **知识库模块**：任务可关联知识卡片
- **仪表盘**：显示今日待办高亮卡片

---

## 7. 非功能需求

- **性能**：支持 500+ 条任务流畅运行
- **数据**：本地优先存储，可导出为 JSON / Markdown
- **提醒**：支持系统通知（桌面版）
- **主题**：完全遵循 Quantum Glass 风格

---

**开发优先级建议**（MVP）：
1. 列表视图 + 基础 CRUD
2. AI 智能创建与拆解
3. 右侧详情面板
4. 看板视图
5. 日历视图 + 智能总结
