用户要求所有文档类输出（架构说明、技术文档等）写成 `.md` 文件保存到项目中，不要直接在控制台输出。

**Why:** 控制台输出不适合阅读和存档长文档。
**How to apply:** 当用户要求生成架构图、技术说明、模块设计等文档类内容时，直接写入 `.md` 文件（通常放在 `docs/` 目录下），控制台只告知文件路径。
