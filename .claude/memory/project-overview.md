HourMind 定位为"私人第二大脑 + 多模型 AI 助理"，支持多厂商 API Key 统一管理、流式对话、知识库、待办事项等功能。

**Why:** 这是一个个人项目，MVP 阶段为纯 Web 应用（localhost），当前使用 SQLite，2026-06-01 后端从 Node.js 切换到纯 Python FastAPI。
**How to apply:** 开发时遵循 CLAUDE.md 中的项目约定，每行代码都需写注释。技术栈：Vue 3 + TypeScript + Pinia / Python 3.12 + FastAPI / sqlite3 / REST + WebSocket。
