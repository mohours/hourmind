# User — 用户画像

## 基本信息
- 外貌：玉树临风
- 角色：全栈开发 + 产品设计
- 背景：Java 程序员，正在学习 Python / Vue / WebSocket
- 语言：中文母语

## 技术栈
- 后端：Java（Spring Boot, MyBatis-Plus）、Python（FastAPI）
- 前端：Vue 3, TypeScript, Pinia
- 数据库：MySQL, SQLite
- 工具：Git, Maven, npm

## 代码风格
- 每行代码必有中文注释，注释说明"为什么"而非"是什么"
- 方案选择倾向简洁——能用原生不用框架，能用简单不用复杂（例：sqlite3 手写 SQL 替代 SQLAlchemy ORM）
- 不喜欢过渡抽象和中间层，代码要直白可读
- 删除无用代码和机制，保持项目精简

## UI 审美
- 酷爱科技感流光溢彩风格——渐变边框、辉光动效、深色主题、玻璃拟态
- 滚动条要定制为渐变色（蓝紫系），不要系统默认样式
- 按钮要有流光渐变 + 悬浮辉光 + 弹性缩放效果
- 字体倾向科技风格（Chakra Petch 等），放弃通用字体（Inter、Arial）

## 沟通偏好
- 用 Java 概念类比解释新技术
- 先给核心结论，再给具体方案，最后指出风险
- 做改动前先讨论方案确认，不要直接动手
- 回答简洁，不啰嗦，不重复说明
- 大功能先出开发计划，用户审阅后再动手

## 交付习惯
- 架构/文档类输出写 md 文件，不直接打印到控制台
- 方案变动后同步更新相关文档（CLAUDE.md、rules 文件等）

## 习惯
- 工作时间：8:30 - 17:30
- 结束信号："下班了"、"再见"、"结束"
