# HourMind - 智能对话 详细产品需求文档（PRD）

## 1. 页面基本信息

**页面名称**：智能对话（Chat / 会话界面）  
**页面定位**：HourMind 的**核心功能页面**，用户使用频率最高的主战场  
**核心目标**：提供沉浸式、高效、强大且具有科技感的个人AI对话体验，支持多模型统一调用，打造“私人专属超级助理”感觉。

**设计风格**：Quantum Glass（量子玻璃）—— 极致玻璃拟态 + 轻科幻 + 高端简约

---

## 2. 用户价值

- 统一入口管理所有AI模型，无需在多个平台切换
- 拥有长期记忆和个性化理解能力
- 极致的交互流畅度和视觉享受
- 支持复杂任务处理（思考、工具调用、知识检索）

---

## 3. 核心功能需求

### 3.1 会话管理
- 新建会话、删除会话、命名/重命名会话
- 会话置顶、星标、归档
- 会话列表搜索（支持按标题、内容、模型、时间搜索）

### 3.2 对话交互
- **多模型实时切换**：顶部快速切换 Claude 4、GPT-4o、DeepSeek、Gemini、Grok 等
- **上下文记忆**：当前会话支持超长上下文（可手动设置长度）
- **全局记忆**：跨会话长期记忆（可通过指令“记住……”实现）
- **流式输出**：支持打字机效果，可中途停止生成
- **Markdown 完美渲染**：支持代码块（高亮+复制）、表格、LaTeX、图片等

### 3.3 消息操作
- 复制消息、重新生成、继续生成
- 将消息保存到知识库
- 点赞/点踩（用于收集反馈）
- 编辑用户已发送消息（可重新生成回复）

### 3.4 输入能力
- 大玻璃输入框（支持多行，自动高度）
- 文件/图片上传（支持拖拽）
- 语音输入（Whisper，后续实现）
- 快捷指令：@知识库、#工具、/命令模式

### 3.5 工具调用（Agent能力）
- 支持工具调用（网页搜索、代码执行、文件操作等）
- 可视化工具调用过程（显示正在使用哪个工具）
- Agent 模式（“进入思考模式”或“帮我完成一个完整任务”）

---

## 4. 页面布局与 UI 细节（Quantum Glass）

**整体布局**：三栏结构
- **左侧导航栏**：常驻（可折叠/悬停呼出）
- **中间聊天区**：占主要空间
- **右侧辅助面板**：默认收起，可展开

**详细设计**：

**顶部栏**（高度 64px）：
- 会话标题（可点击编辑）
- 当前模型标签（玻璃样式，显示模型名 + 提供商图标）
- 操作按钮：新建会话、保存到知识库、右侧面板开关、更多菜单

**消息区域**：
- AI 消息：左对齐，玻璃背景 + 左侧量子青细竖条 + 小头像
- 用户消息：右对齐，较深量子青玻璃背景
- 空状态：中间显示量子 Logo + 欢迎语 + 4 个推荐问题玻璃卡片

**底部输入栏**（固定）：
- 极致玻璃拟态输入框
- 左侧工具按钮：上传、语音、@知识库、工具
- 右侧：模型快速切换 + 发送按钮（带辉光）

**右侧辅助面板**（280px）：
- 当前会话信息（Token 使用、创建时间）
- 本次会话关键记忆点
- 关联知识库推荐
- AI 思考链/工具调用记录

---

## 5. 交互与动效

- **消息出现**：底部淡入 + 轻微上浮
- **流式输出**：逐字显示 + 底部微弱量子光条扫描（可关闭）
- **模型切换**：平滑过渡 + 短暂辉光效果
- **输入框聚焦**：边框辉光增强
- **快捷键**：
  - Ctrl + N：新建会话
  - Ctrl + K：切换模型
  - Ctrl + Shift + K：打开知识库检索
  - Esc：停止生成

---

## 6. 与其他模块的关联

- **知识库**：对话中可直接引用知识库内容，或将对话保存至知识库
- **待办事项**：对话中可直接创建任务（“把这个做成待办”）
- **API Key 管理**：无缝调用已配置的模型
- **仪表盘**：最近对话卡片可直接跳转到此页面

---

## 7. 非功能需求

- **响应速度**：输入后 1 秒内开始回复（流式）
- **移动端**：支持响应式布局（重点优化输入体验）
- **隐私**：所有对话数据本地/自有服务器存储
- **可扩展性**：为后续多模态（语音、图像、视频）预留接口

---

## 8. 开发优先级（MVP）

**Phase 1（核心）**：
1. 基础对话收发 + 流式输出
2. 多模型切换
3. 会话管理（新建、切换、删除）
4. Markdown 渲染 + 消息操作

**Phase 2**：
5. 文件上传 + 右侧辅助面板
6. 全局记忆功能
7. 工具调用框架

**Phase 3**：
8. Agent 模式 + 高级指令
9. 语音输入
