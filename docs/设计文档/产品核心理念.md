1. 产品核心理念
HourMind 是一个真正属于你自己的智能系统，它不是简单的 ChatGPT 包装器，而是：

你个人的知识大脑 + 生活助理 + 工作工具的结合体
支持多模型统一管理（Claude 最强时用 Claude，速度需求高时用 DeepSeek）
拥有持久记忆和个性化能力
界面简洁现代，操作尽可能丝滑


2. 核心功能模块（详细版）
模块一：智能对话（核心中的核心）

多模型即时切换（下拉选择或快捷键）
会话管理（新建会话、命名、删除、置顶）
上下文记忆（当前会话支持很长的上下文）
全局记忆（跨会话长期记忆，你可以告诉它“记住我喜欢用深色模式”或“我的常用编程语言是Java和Python”）
流式输出 + Markdown 渲染 + 代码高亮
一键复制、重新生成、继续书写
支持语音输入（可选，后续添加）

模块二：API Key 管理（非常重要）

支持主流厂商：OpenAI、Anthropic（Claude）、DeepSeek、Gemini、Grok、Moonshot、通义千问等
添加 Key 后可立即测试连通性
Key 别名 + 使用场景标签（例如：Claude-主力、DeepSeek-速度）
用量统计（每日/每月 Token 消耗和预估费用）
智能路由规则（可自定义：复杂思考用 Claude、日常问答用 DeepSeek）

模块三：个人知识库（Second Brain）

上传各种文件（PDF、Word、Markdown、TXT、图片等）
自动向量化 + 智能总结
知识检索对话（可指定“只基于我的知识库回答”）
笔记系统（支持快速记录想法并自动关联）
知识图谱浏览（后续增强功能）

模块四：工具与自动化

基础工具：网页搜索、实时信息获取、代码解释器
个人工具：读写本地文件（需授权）、操作日历、发送邮件/微信通知
自动化：创建简单定时任务（如“每天早上8点总结昨天重点事项”）
Agent 模式（“帮我完成一个完整任务”）

模块五：仪表盘与设置

今日/本周使用概览
模型使用分布图
快速入口（常用会话、知识库、设置）
个性化设置（主题、默认模型、回复风格、记忆强度等）


3. 用户使用流程示例

打开 HourMind → 看到仪表盘
点击聊天 → 选择模型 → 开始对话
对话中可以随时切换模型、上传文件、调用工具
重要对话可保存到知识库
在设置中管理所有 API Key 和路由规则


4. 特色与差异化

多模型统一体验：不用在不同网站切换
强记忆能力：比普通 ChatGPT 更“懂你”
隐私优先：所有数据本地或自己服务器存储
可玩性高：容易扩展新工具和能力
学习成长：系统会随着你使用越来越聪明
