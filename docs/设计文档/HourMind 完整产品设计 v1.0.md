HourMind 完整产品设计 v1.0
主题：Quantum Glass（量子玻璃）
核心理念：安静、高端、专注的个人量子智能空间

1. 整体布局结构
采用 三栏布局（左侧导航 + 中间主内容 + 右侧辅助面板）：

左侧导航（固定 260px）
中间内容区（自适应）
右侧面板（280px，可折叠）—— 提供上下文信息


2. 主要页面详细设计
页面 1：仪表盘（Dashboard） - 首页
布局：

顶部：欢迎语 + 当前日期 + 快速新建对话按钮（量子青大按钮）
中间：卡片网格（响应式 2~3 列）
今日概览卡片：玻璃卡片，显示今日 Token 用量、对话次数、活跃模型（环形图）
快速启动：4 个大玻璃按钮（新对话、上传文件到知识库、创建自动化任务、打开设置）
最近对话：横向滚动卡片列表
知识库状态：已索引文件数 + 容量


科技感细节：

卡片悬停时轻微上浮 + 辉光增强
环形图使用量子青 + 淡紫渐变


页面 2：智能对话（主战场）
布局：

顶部栏：会话标题（可编辑） + 模型切换器（下拉 + 当前模型玻璃标签） + 右侧面板开关
消息区域（占满剩余空间，可滚动）：
AI 消息：左对齐，玻璃背景 + 左侧量子青细条
用户消息：右对齐，较深量子青玻璃背景
支持 Markdown、代码块（带复制）、表格、LaTeX

底部输入区（固定玻璃栏）：
大输入框（支持回车发送，Shift+回车换行）
左侧工具按钮：上传文件、语音输入、@知识库
右侧：发送按钮（量子青发光）


交互亮点：

模型切换实时生效（无需刷新）
长按消息可出现操作菜单（复制、重新生成、保存到知识库）
流式输出时，消息底部有微弱的量子光条扫描动画


页面 3：API Key 管理
布局：

顶部：大按钮 “+ 添加新 API Key”
主体：玻璃卡片列表（每张 Key 一张卡片）
每张卡片包含：
左侧平台 Logo + 名称（OpenAI / Claude 等）
中间：别名、创建时间、状态标签（正常 / 即将过期 / 失效）
右侧：用量小环形图 + 操作按钮（测试、编辑、删除）


添加 Key 弹窗：

玻璃模态框
选择平台 → 输入 Key → 别名 → 测试按钮（实时返回响应速度和状态）


页面 4：知识库（Second Brain）

左侧文件夹/标签列表
中间文件列表（支持搜索）
右侧预览 + 智能总结
支持拖拽上传
“智能检索” 开关（对话时是否优先使用知识库）


页面 5：设置中心
分类卡片形式：

AI 模型设置（默认模型、路由规则、温度）
外观设置（粒子背景开关、辉光强度、字体）
数据与隐私（导出全部数据、清空记忆、加密设置）
关于系统


3. 交互与动效设计

全局一致：
所有卡片 hover：轻微上浮（transform: translateY(-4px)）+ 辉光增强
按钮点击：轻微按压 + 量子青光效
页面切换：淡入过渡（0.25s）

特色动效：
模型切换时，当前模型标签有“量子跃迁”光效
新消息出现时，从底部淡入 + 轻微振动
加载状态使用量子青的扫描线动画
