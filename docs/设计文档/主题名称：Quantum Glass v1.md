主题名称：Quantum Glass v1.0
风格总结：极致玻璃拟态 + 轻科幻 + 高端简约 + 安静未来感

1. 全局配色方案
元素  颜色代码  使用场景  
主背景  #0A0C12 整个应用背景
玻璃卡片背景  rgba(16, 18, 27, 0.72)  所有卡片、侧边栏、对话框
主强调色  #00E5D8 (量子青)  按钮、选中状态、辉光
辅助强调色  #C4B5FD (淡紫)  次要按钮、标签、链接
文字主色  #F1F5F9 标题、正文
文字次色  #94A3B8 辅助文字、提示
边框/辉光 rgba(0, 229, 216, 0.25) 玻璃边框
成功色  #34D399 测试成功
警告色  #FB923C 警告、注意
暗黑模式：强制使用（不提供亮色版本，保持科技纯粹感）

2. 全局样式规则

玻璃效果统一 CSS
```
.glass {
  background: rgba(16, 18, 27, 0.72);
  backdrop-filter: blur(20px);
  border: 1px solid rgba(0, 229, 216, 0.18);
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.3),
              0 0 0 1px rgba(0, 229, 216, 0.1) inset;
}
```
字体：
正文：Inter 或系统 -apple-system
标题：Inter Bold + 适当 letter-spacing

圆角：统一 16px（高端柔和感）


3. 主要页面布局与风格
A. 整体布局（三栏结构）

左侧导航栏（宽度 260px）
顶部 Logo（文字 + 小量子图标）
导航菜单（仪表盘、对话、知识库、工具、设置）
底部：当前模型状态 + 用户信息

中间主内容区（自适应）
聊天页面、仪表盘等主要内容

右侧辅助栏（宽度 280px，可折叠）
当前会话信息
记忆/知识库快速引用
Token 使用情况


B. 聊天主界面（最重要）

顶部：会话标题 + 模型切换下拉（带量子青发光）
中间：消息区域
用户消息：右对齐，量子青玻璃背景
AI 消息：左对齐，深玻璃背景 + 左侧细紫线

底部：固定输入框（大玻璃栏）
支持上传文件、语音输入、模型快速切换

流式输出时，AI 消息区域有柔和的量子青光条扫描效果

C. 仪表盘首页

顶部欢迎语：“早上好，小时。今天想做什么？”
卡片网格（3列）：
今日使用概览（环形图）
常用模型
最近对话
知识库状态


D. API Key 管理页面

顶部大按钮：“添加新 API Key”
Key 以玻璃卡片形式垂直排列
每张卡片包含：平台图标、别名、状态（正常/失效）、用量
测试按钮 hover 时出现量子扫描动画
