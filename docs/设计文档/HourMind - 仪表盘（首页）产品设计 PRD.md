# HourMind - 仪表盘（首页）产品设计 PRD

## 1. 功能概述

**页面名称**：仪表盘（Dashboard / 首页）  
**页面定位**：HourMind 的核心入口和全局概览页  
**核心目标**：让用户一打开应用就能快速了解系统状态、今日重点、快速开始常用操作，打造沉浸式科技感首页。

**设计风格**：严格遵循 **Quantum Glass（量子玻璃）** 主题  
- 深空背景 + 玻璃拟态卡片 + 量子青 `#00E5D8` 强调色

---

## 2. 用户价值

- 一目了然地看到今日/本周使用情况
- 快速进入最常用的功能
- 获得 AI 的个性化欢迎与建议
- 建立“这是我自己的智能空间”的归属感

---

## 3. 页面布局结构

**整体采用三栏布局**（与聊天页面保持一致）：

- **左侧导航栏**：常驻（与整体系统统一）
- **中间主内容区**：卡片网格布局（响应式）
- **右侧辅助面板**：可选折叠，显示今日重点推荐

---

## 4. 详细模块设计

### 4.1 顶部区域
- **欢迎语**：个性化文案，例如 “早上好，小时” / “晚上好，今天已经和 Claude 对话 12 次”
- 当前日期 + 星期
- 快速新建按钮（大量子青按钮）：**新对话**、**新建任务**、**上传文件**

### 4.2 核心指标卡片（网格布局）

**第一行（大卡片）**：
- **今日使用概览**（占两列）
  - Token 消耗环形图（今日 / 本月）
  - 对话次数、任务完成数
  - 活跃模型排行

- **AI 今日建议**（玻璃卡片）
  - AI 根据历史使用习惯生成的个性化建议（如“今天可以继续完善上周的项目计划”）

**第二行**：
- **快速入口卡片**（4 个等大玻璃卡片）
  - 智能对话
  - 个人知识库
  - 个人待办事项
  - 工具中心（后续扩展）

- **最近对话**（横向滚动玻璃卡片）
  - 显示 4-6 条最近会话
  - 点击可直接打开对应会话

- **今日待办**（精简列表）
  - 显示 3-5 条最高优先级待办
  - 支持快速勾选完成

- **知识库状态**
  - 已索引文件数
  - 本月新增知识
  - 容量使用情况

---

## 5. 交互与动效

- **卡片悬停**：轻微上浮 + 辉光增强
- **环形图**：量子青 + 淡紫渐变，加载时有扫描动画
- **欢迎语**：根据时间段、用户习惯动态变化
- **点击快速入口**：平滑跳转到对应页面
- **刷新机制**：页面加载或切换回来时自动更新数据

---

## 6. 与其他模块关联

- 点击**最近对话** → 跳转到对应聊天页面
- 点击**今日待办**中的任务 → 跳转到个人待办事项页面（并定位到该任务）
- 点击**知识库状态** → 跳转到个人知识库
- 所有数据实时同步（通过 WebSocket）

---

## 7. 非功能需求

- **加载性能**：首屏加载时间 < 1.5 秒
- **移动端适配**：支持响应式布局（平板可用）
- **个性化**：根据用户使用习惯动态调整卡片优先级（后续）
- **数据来源**：从后端 API 统一获取

---

## 8. 开发优先级（MVP）

1. 基础布局 + 静态卡片
2. 今日使用概览（Token、对话次数）
3. 快速入口 + 最近对话
4. 今日待办精简列表
5. AI 今日建议（简单版本）
