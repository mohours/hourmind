# HourMind - 系统设置 产品需求文档（PRD）

## 1. 页面基本信息

**页面名称**：系统设置（Settings）
**页面定位**：HourMind 的全局配置中心
**核心目标**：让用户一站式管理 AI 参数、外观偏好、数据安全和系统信息。

**设计风格**：Quantum Glass（量子玻璃）—— 分类玻璃卡片布局。

---

## 2. 用户价值

- 统一入口管理所有系统配置
- 个性化 AI 回复风格（温度、最大 Token）
- 自定义界面视觉（粒子背景、辉光强度）
- 数据掌握在自己手里（导出、清除、加密）

---

## 3. 核心功能需求

### 3.1 AI 模型设置

- **默认模型**：下拉选择，联动已添加的 API Key 可用模型列表
- **模型温度**：滑块 0~2（默认 0.7），实时显示当前值
- **最大 Token 数**：数字输入框（默认 4096）
- **上下文长度**：数字输入框（默认 32000）
- **联网搜索**：默认开关（开启/关闭）

### 3.2 外观设置

- **粒子背景**：开关（开启时深空背景显示微弱的浮动粒子动画）
- **辉光强度**：三档切换 low / medium / high（影响卡片 hover 辉光和边框亮度）
- **消息字体大小**：三档 small / medium / large（默认 medium）
- **动画速度**：三档 fast / normal / slow（影响过渡和消息入场动画）

### 3.3 数据与隐私

- **导出全部数据**：一键导出为 JSON（含对话、任务、知识库）
- **清除所有对话**：二次确认弹窗，输入"确认删除"才执行
- **清除所有 Key**：二次确认弹窗
- **重置系统**：恢复出厂设置，删除所有数据（密码也重置）

### 3.4 关于系统

- **版本号**：v1.0.0
- **技术栈信息**：Vue 3 + FastAPI + SQLite
- **GitHub 链接**：项目仓库地址（如有）
- **检查更新**：按钮（后续版本实现）

---

## 4. 页面布局（Quantum Glass）

**整体布局**：单栏主内容区

**页面结构**：

```
顶部标题：系统设置
├── AI 模型设置卡片
│   ├── 默认模型（下拉）
│   ├── 模型温度（滑块 | 当前值: 0.7）
│   ├── 最大 Token 数（数字输入框）
│   ├── 上下文长度（数字输入框）
│   └── 联网搜索默认开关
├── 外观设置卡片
│   ├── 粒子背景（开关）
│   ├── 辉光强度（三档切换按钮组）
│   ├── 消息字体大小（三档切换按钮组）
│   └── 动画速度（三档切换按钮组）
├── 数据与隐私卡片
│   ├── 导出全部数据（按钮）
│   ├── 清除所有对话（危险按钮 | 红色警告）
│   ├── 清除所有 Key（危险按钮 | 红色警告）
│   └── 重置系统（危险按钮 | 红色警告 + 二次确认输入）
└── 关于卡片
    ├── 版本号
    ├── 技术栈
    └── 检查更新
```

**交互细节**：
- 每个设置项即时生效（或显示"已保存"提示）
- 危险操作按钮使用警告色（`#FB923C` 或红色），点击后弹出确认弹窗
- 开关使用量子青 `#00E5D8` 配色
- 滑块 hover 显示当前数值

---

## 5. 交互与动效

- **设置变更**：滑块松开或失去焦点后自动保存，卡片右下角短暂显示"✓ 已保存"
- **开关切换**：量子青扩散动画
- **危险操作**：二次确认模态框，背景加暗，需点击确认或输入文字
- **导出数据**：完成后按钮变为"✓ 导出成功"，3 秒后恢复
- **上下按钮组**：选中项有量子青背景 + 深色文字

---

## 6. 与其他模块关联

- **API Key 管理**：默认模型列表来自于已添加且状态正常的 Key
- **智能对话**：温度、最大 Token、上下文长度直接影响对话行为
- **仪表盘**：外观设置影响全局 UI 效果
- **所有页面**：粒子背景和辉光强度全局生效

---

## 7. 非功能需求

- **即时保存**：设置变更后 500ms 内写入数据库
- **数据导出**：生成 JSON 文件并触发浏览器下载
- **安全确认**：所有不可逆操作必须二次确认
- **设置持久化**：页面刷新后设置不丢失

---

## 8. 开发优先级（MVP）

1. AI 模型设置（默认模型 + 温度 + 最大 Token）
2. 外观设置（粒子背景 + 辉光强度）
3. 数据导出 + 清除
4. 关于系统
5. 上下文长度 + 消息字体 + 动画速度（锦上添花）
