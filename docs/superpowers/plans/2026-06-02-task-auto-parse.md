# 待办智能解析 — 开发计划

**Goal:** 粘贴一段文本 → 点击"智能解析" → AI 自动提取标题/描述/优先级/日期 → 填充表单

## 涉及文件

| 文件 | 改动 |
|------|------|
| `server/routes/tasks.py` | 新增 `POST /api/tasks/parse` 端点 |
| `client/src/views/TasksView.vue` | 添加"粘贴解析"区域 |

## Task 1: 后端解析端点

`POST /api/tasks/parse`，需认证。

**请求：**
```json
{ "text": "【部门周例会通知】\n时间：2026年5月29日..." }
```

**响应：**
```json
{
  "title": "部门周例会",
  "description": "时间：2026年5月29日...\n会议议题：...",
  "priority": "high",
  "due_date": "2026-05-29T08:45:00"
}
```

**实现逻辑：**
1. 查活跃 API Key → 解密
2. 构造 prompt 发给 AI："将以下文本解析为待办任务，返回 JSON..."
3. 要求 AI 返回固定 JSON 格式（系统提示词约束），用正则提取 `{...}`
4. 返回解析结果

**prompt 设计：**
```
你是一个任务解析助手。分析用户输入的文本，提取关键信息，返回以下 JSON 格式（只返回 JSON，不要其他内容）：
{"title":"任务标题","description":"详细描述","priority":"low/medium/high/urgent","due_date":"ISO日期或null"}

优先级判断：包含"紧急/截止/必须/周会"等 → high，日常任务 → medium，不重要 → low
日期提取：识别"本周五/明天/2026年5月29日/8:45"等，转为 ISO 格式
```

## Task 2: 前端粘贴解析

在 TasksView 创建/编辑弹窗中：

1. **粘贴区** — 标题字段上方加一个大 textarea（placeholder: "粘贴文本，自动解析为待办..."），初始折叠
2. **"智能解析"按钮** — textarea 下方，调用 `/api/tasks/parse`
3. **自动填充** — 解析成功后自动填充 title、description、priority、due_date
4. **loading 状态** — 按钮显示"解析中..."

## 交互流程

```
用户点击"添加任务" → 弹窗打开
  → 顶部有"粘贴文本"折叠区（默认展开）
  → 粘贴会议通知文本
  → 点击"智能解析 ✨"
  → 按钮显示"解析中..."
  → 收到结果 → 自动填入标题/描述/优先级/日期
  → 用户确认/修改 → 点击确认添加
```

## 验收标准

- [ ] 粘贴会议通知文本 → 解析出标题"部门周例会"
- [ ] 优先级自动识别为 high（含"周会"关键词）
- [ ] 日期解析为 2026-05-29T08:45
- [ ] 没有 API Key 时提示"请先添加 API Key"
- [ ] 解析失败时显示错误提示，不覆盖已有表单数据
