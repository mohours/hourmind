# 待办智能解析 — 设计文档

**Goal:** 粘贴任意文本 → AI 自动提取标题/优先级/日期 → 原文作为描述 → 一键创建待办

## 架构

```
前端 TasksView 弹窗                   后端
  ┌─────────────────┐           ┌──────────────────┐
  │ 粘贴区 textarea  │           │ POST /api/tasks/  │
  │      ↓           │── AI ──→ │      parse        │
  │ "智能解析" 按钮   │←─ JSON ─│                   │
  │      ↓           │           │ 调 AI 返回 {title,│
  │ 自动填表单各字段   │           │  priority, date} │
  └─────────────────┘           └──────────────────┘
```

## 后端：POST /api/tasks/parse

**输入：** `{ "text": "粘贴的原始文本" }`

**AI Prompt：**
```
分析以下文本，提取任务标题、优先级和截止日期。返回纯 JSON：
{"title":"任务标题","description":"详细描述","priority":"low/medium/high/urgent","due_date":"ISO格式日期或null"}

优先级判断规则：
- 含"紧急/立即/截止/马上" → urgent
- 含"会议/周会/汇报/重要/必须" → high
- 普通任务 → medium
- 无明确时限 → low
```

**输出：** `{ "title":"部门周例会", "priority":"high", "due_date":"2026-05-29T08:45:00" }`

## 前端：TasksView 弹窗改动

在"标题"字段上方插入粘贴区：

```
┌─ 添加任务 ──────────────────────┐
│                                  │
│  📋 智能解析（新增大块）           │
│  ┌──────────────────────────┐   │
│  │ 粘贴文本，自动识别待办...   │   │
│  │                          │   │
│  └──────────────────────────┘   │
│  [  智能解析 ✨  ]               │
│                                  │
│  标题：   [部门周例会        ]   │
│  描述：   [粘贴的原文...     ]   │
│  优先级： [高 ▼]                │
│  日期：   [2026-05-29      ]    │
│                                  │
│  [取消]          [确认添加]      │
└──────────────────────────────────┘
```

**交互：**
1. 粘贴文本 → 点击"智能解析"
2. 按钮变 "解析中..." disabled
3. 成功 → 自动填 title/priority/due_date，描述填原文
4. 失败 → 显示红色错误提示，不覆盖已有内容
5. 无 API Key → 提示"请先在 Key 管理中添加 API Key"
