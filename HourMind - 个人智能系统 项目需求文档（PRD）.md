# HourMind（小时智脑） - 个人智能系统 完整产品需求文档（PRD）

## 1. 项目基本信息

**项目名称**：HourMind（小时智脑）  
**版本**：v1.0  
**定位**：属于个人的高端智能系统（私人第二大脑 + 多模型智能助理）  
**核心目标**：提供统一、强大、可控且具有科技感的个人AI使用体验  
**目标用户**：个人长期使用（单用户优先）

**UI风格**：**Quantum Glass（量子玻璃）**  
- 极致玻璃拟态 + 轻科幻 + 高端简约  
- 主色调：量子青 `#00E5D8` + 深空背景 `#0A0C12`

---

## 2. 核心功能模块

### Phase 1 - MVP（必须实现）

1. **智能对话系统**（核心）
   - 多模型统一切换（Claude、GPT-4o、DeepSeek、Gemini、Grok等）
   - 会话管理（新建、命名、删除、置顶）
   - 长上下文记忆
   - 流式输出 + Markdown 完美渲染
   - 消息操作（复制、重新生成、保存到知识库）

2. **API Key 管理**
   - 支持主流厂商 API Key 添加、测试、编辑、删除
   - Key 安全加密存储（仅后端）
   - 用量统计与成本预估
   - 模型智能路由规则

3. **仪表盘（首页）**
   - 今日使用概览
   - 快速入口卡片
   - 最近会话
   - 系统状态

4. **历史记录与搜索**
   - 所有对话历史
   - 关键词搜索

5. **系统设置**
   - 外观（粒子背景、辉光强度等）
   - AI 参数设置
   - 数据管理

### Phase 2 - 后续迭代（建议）

- 个人知识库（RAG + 文件上传）
- 工具调用（网页搜索、代码执行等）
- 自动化任务与 Agent 模式
- 语音输入/输出
- 多用户支持

---

## 3. UI/UX 设计规范（Quantum Glass）

**全局风格**：
- 深色强制模式
- 玻璃拟态（Glassmorphism）：半透明 + backdrop blur + 细辉光边框
- 主色：量子青 `#00E5D8`
- 圆角：16px
- 字体：Inter（标题可适当加粗 + letter-spacing）

**主要页面风格**：

**聊天主界面**（最重要）：
- 三栏布局（左侧导航 + 中间聊天 + 可折叠右侧面板）
- 顶部：会话标题 + 模型切换标签
- 消息区：AI 左（玻璃 + 青色竖条）、用户右（量子青玻璃）
- 底部：大玻璃输入栏 + 工具按钮

**仪表盘**：
- 欢迎语 + 模块化玻璃卡片网格

**API Key 管理**：
- 玻璃卡片列表形式，每张 Key 为独立悬浮面板

**动效原则**：
- 轻柔、克制、有科技感（淡入、上浮、辉光呼吸）
- 可在设置中调节动画强度

---

## 4. 技术栈（已确认）

- **前端**：Vue 3 + TypeScript + Vite + Element Plus + Pinia
- **后端**：Node.js + Express + TypeScript
- **AI层**：Python + FastAPI + LiteLLM
- **数据库**：SQLite（初期） / PostgreSQL + PGVector
- **其他**：Prisma、Socket.io、Docker

---

## 5. 非功能需求

- **隐私安全**：API Key 绝不暴露到前端，所有敏感数据加密存储
- **性能**：个人项目，优先流畅度和开发体验
- **扩展性**：代码结构清晰，便于后续添加知识库和 Agent 功能
- **学习性**：便于你学习 Node.js + TypeScript 全栈开发

---

**开发优先级建议**：

1. 项目基础框架（前后端 + 通信）
2. API Key 管理模块（含测试功能）
3. 聊天主界面 + 流式对话
4. 仪表盘与历史记录
5. 设置中心
6. 知识库（Phase 2）
